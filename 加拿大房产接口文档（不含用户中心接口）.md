### 接口
#### 测试地址
#### API版本：
##### v0.1
#### API规范
##### Header请求头部拼装
    {
        "Authorization": "" // token
        "User-Agent": "" // 设备信息
    }
##### Response返回实体
    {
        "status": Int // 状态 0-失败 1-成功
        "descript": String  // 状态描述
        "totalCount": Int // 列表总记录数
        "data" // 数据实体或集合
    }
